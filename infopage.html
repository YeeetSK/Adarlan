Server je vytvořen na způsob Semi-Vanilla.
Zde najdeš několik možností aktivit, kterou můžeš vykonávat buď sám nebo s ostatními spoluhráči, ale ať už si zvolíš možnost první či druhou určitě tě to nepřestane bavit a jentak pro tebe tento server neskončí, protože má v zásobě spoustu nápadů, co se budou realizovat postupem času. A čím je jiný od ostatních serverů? Server je založen na kvalitním hostingu se svou doménou. Server bude mít téma, který se nazývá EVOLVE.
O čem to je?
Týdenní hlasování o vylešpení Vanilly. Stačí hlasovat každý týden o jeden ze tří vylepšení. Každou středu bude odhlasováno jedno vylepšení, které bude přidáno!
